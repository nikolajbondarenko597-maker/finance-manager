# Personal Finance Manager

Backend-приложение для управления личными финансами с поддержкой регистрации, бюджетов, статистики и автоматического сохранения данных.

## Функции
- Регистрация и авторизация пользователей
- Добавление доходов и расходов
- Установка бюджетов по категориям
- Просмотр отчёта с анализом трат
- Автоматическое сохранение данных при выходе
- Оповещение о превышении бюджета или перерасходе

## Требования
- Java 17+
- Maven

## Запуск
1. Соберите проект:
   ```bash
   mvn compile
   ```
2. Запустите:
    ```
   mvn exec:java -Dexec.mainClass="com.promoit.finance.cli.FinanceApp"
   ```
### Или
Или просто нажмите зелёную стрелку в FinanceApp через IDEA.

## Хранение данных
Все данные пользователей сохраняются в папке data/ в формате JSON.
При входе данные автоматически загружаются.

## Тестирование
### Запустите тесты:
```

```
